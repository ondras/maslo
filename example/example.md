# Keyboard reference

- `Space` / `PageDown` / &rarr; / &darr; : next slide
- `Backspace` / `PageUp` / &larr; / &uarr; : previous slide
- `Home`: first slide
- `End`: last slide
- `ESC`: toggle overview
- `Caps Lock`: toggle doodling
